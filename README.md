🚀 CRED Clone – React Frontend Project
📌 Overview
This project is a frontend clone of the CRED website, built using React.js + Vite.
It replicates CRED’s premium UI/UX experience with a dark theme, NeoPOP design system, smooth animations, and fully responsive layout.
This project is created for improving frontend development skills, UI replication accuracy, and animation handling in React.
🎯 Features

* ⚡ Fully responsive landing page (mobile, tablet, desktop)
* 🎨 Premium dark UI inspired by CRED
* ✨ Smooth animations and transitions
* 🧠 Component-based React architecture
* 📱 Mobile-first responsive design
* 💎 NeoPOP-style UI (glassmorphism, shadows, glow effects)
* 🖱️ Interactive hover effects & micro-interactions
* 🎬 Smooth section scrolling experience
🛠️ Tech Stack

* React.js (Vite)
* HTML5
* CSS3
* JavaScript (ES6+)
* CSS3 Animations (keyframes)
* IntersectionObserver API

🚀 Getting Started
Follow these steps to run the project locally:

```
# Clone the repository
git clone https://github.com/itzz-dhivya/cred-clone.git

# Navigate to project folder
cd cred-clone

# Install dependencies
npm install

# Start development server
npm run dev
```
